TLCI - Tiny Lambda Calculus Interpreter
A lambda calculus interpreter in C99. I mostly developed it on my own, but parts of it on Twitch. See the Twitch Playlist.

GitLab

Features


  • - Strict evaluation model
  • - Mark-and-sweep garbage collector
  • - Loading files
  • - Binding/releasing variables in the global scope

Commentary


The goal of this project was to reimplement my previous implementation of a lambda calculus interpreter, which was written in Haskell, in C99 to make it cross-platform.
I wanted to give flex and yacc a shot because I was hacking on Hugs98 and it used those tools. I have never used them before so that was a good reason to do something with'em.
The project itself only depends on a C99 compiler, flex, yacc and make. For readline support I recommend `rlwrap`, a little tool that adds support for readline in applications that don't support it.
I deliberatly didn't add it as a dependency, since I wanted to have as few as possible and also, the GNU libs are usually a bit difficult to work with on foreign platforms.

heat-cascade - Heat Cascade Algorithm in D
GitLab
This was actually something I needed for Uni. Basically this is what this little program does: it calculates the minimum required amount of heat and cooling energy that has to be supplied in a network of heat exchangers inside a e.g. chemical plant to save on cost when running a process. It takes in the streams of liquids or gases and a minimum temperature difference. Based on this data, it spits out the optimal values for QH and QK, the minimum heating energy and the minimum cooling energy.
